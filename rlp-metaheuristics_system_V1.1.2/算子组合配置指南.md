# 算子组合配置指南

本文档详细说明了如何指定和修改算法的算子组合配置。

---

## 📊 算子组合统计

**总组合数：225种**

| 算法 | 组合数 | 详细说明 |
|------|--------|---------|
| **DE** | **80种** | 10种变异策略 × 4种交叉策略 × 2种局部搜索 |
| **GA** | **128种** | 2种选择 × 4种交叉 × 2种变异 × 2种初始化 × 2种局部搜索 × 2种邻域 |
| **SA** | **1种** | 默认配置 |
| **ILS** | **5种** | 5种扰动强度（3, 5, 7, 10, 15） |
| **TS** | **2种** | 2种禁忌策略（static, dynamic） |
| **PR** | **8种** | 2种路径策略 × 2种选择策略 × 2种局部搜索 |
| **GSA** | **1种** | 默认配置 |

## 🎯 方式2：创建自定义配置函数

### 步骤1：创建自定义配置文件

参考 `custom_algorithm_configs.py` 文件，创建您自己的配置函数：

```python
def generate_custom_algorithm_configs():
    """生成自定义的算子组合配置"""
    configs = []
    
    # ========================================
    # 示例1：只运行DE算法的特定变异策略
    # ========================================
    de_mutation_strategies = [
        ("rand/1", True),   # rand/1 with adaptive F
        ("best/1", True),   # best/1 with adaptive F
    ]
    
    de_crossover_strategies = [
        ("bin", True),      # bin with adaptive CR
        ("exp", False),     # exp with fixed CR
    ]
    
    for mut_strat, use_adapt_F in de_mutation_strategies:
        for cross_strat, use_adapt_CR in de_crossover_strategies:
            config_name = f"DE_{mut_strat}_{'adaptiveF' if use_adapt_F else 'fixedF'}_{cross_strat}_{'adaptiveCR' if use_adapt_CR else 'fixedCR'}"
            configs.append((config_name, "DE", {
                "mutation_strategy": mut_strat,
                "use_adaptive_F": use_adapt_F,
                "crossover_strategy": cross_strat,
                "use_adaptive_CR": use_adapt_CR,
                "use_local_search": False,
            }))
    
    # ========================================
    # 示例2：只运行GA算法的特定配置
    # ========================================
    ga_configs = [
        # 配置1：轮盘赌 + 单点交叉 + 随机变异
        {
            "selection_strategy": "roulette",
            "crossover_strategy": "single_point",
            "mutation_strategy": "random",
            "initialization_strategy": "random",
            "use_local_search": True,
            "neighborhood_size": 2,
        },
        # 配置2：锦标赛 + 两点交叉 + 自适应变异
        {
            "selection_strategy": "tournament",
            "crossover_strategy": "two_point",
            "mutation_strategy": "adaptive",
            "initialization_strategy": "heuristic",
            "use_local_search": False,
            "neighborhood_size": 0,
        },
    ]
    
    for i, params in enumerate(ga_configs, 1):
        config_name = f"GA_custom_{i}"
        configs.append((config_name, "GA", params))
    
    return configs
```

### 步骤2：修改runner.py

在 `src/eval/runner.py` 中添加：

```python
# 导入自定义函数
from custom_algorithm_configs import generate_custom_algorithm_configs

# 修改ExperimentRunner的__init__方法
class ExperimentRunner:
    def __init__(self, config: ExperimentConfig):
        self.config = config
        self.results: List[RunResult] = []
        
        # 使用自定义配置
        if "custom" in self.config.algorithms:
            self.all_configs = generate_custom_algorithm_configs()
        else:
            self.all_configs = generate_all_algorithm_configs()
```

### 步骤3：在main.py中使用

```python
algorithms=["custom"]  # 使用自定义配置
```

## 📋 算子组合命名规则

### DE算法命名规则

```
DE_{变异策略}_{F类型}_{交叉策略}_{CR类型}_{局部搜索}
```

**示例**：
- `DE_rand/1_adaptiveF_bin_adaptiveCR_withLS`
- `DE_best/1_fixedF_exp_fixedCR_noLS`

**组成部分**：
- **变异策略**：rand/1, rand/2, best/1, best/2, adaptive, current-to-rand/2
- **F类型**：adaptiveF（自适应F）, fixedF（固定F）
- **交叉策略**：bin, exp
- **CR类型**：adaptiveCR（自适应CR）, fixedCR（固定CR）
- **局部搜索**：withLS（启用）, noLS（禁用）

### GA算法命名规则

```
GA_{选择策略}_{交叉策略}_{变异策略}_{初始化策略}_{局部搜索}_{邻域}
```

**示例**：
- `GA_roulette_single_point_random_random_withLS_withNeighborhood`
- `GA_tournament_two_point_adaptive_heuristic_noLS_noNeighborhood`

**组成部分**：
- **选择策略**：roulette, tournament
- **交叉策略**：single_point, two_point, rcx, hybrid
- **变异策略**：random, adaptive
- **初始化策略**：random, heuristic
- **局部搜索**：withLS（启用）, noLS（禁用）
- **邻域**：withNeighborhood（启用）, noNeighborhood（禁用）

### 其他算法命名规则

**SA算法**：
- `SA_default`

**ILS算法**：
- `ILS_strength3`
- `ILS_strength5`
- `ILS_strength7`
- `ILS_strength10`
- `ILS_strength15`

**TS算法**：
- `TS_static`
- `TS_dynamic`

**PR算法**：
- `PR_forward_best_withLS`
- `PR_forward_best_noLS`
- `PR_forward_random_withLS`
- `PR_forward_random_noLS`
- `PR_backward_best_withLS`
- `PR_backward_best_noLS`
- `PR_backward_random_withLS`
- `PR_backward_random_noLS`

**GSA算法**：
- `GSA_default`

## 🔍 常见问题

### Q1: 如何查看所有可用的算子组合名称？

**A**: 运行以下命令：

```bash
python -c "
from src.eval.runner import generate_all_algorithm_configs
configs = generate_all_algorithm_configs()
for config_name, algo_type, params in configs:
    print(config_name)
"
```

### Q2: 如何只运行自适应参数的组合？

**A**: 使用关键词过滤：

```python
algorithms=["adaptiveF", "adaptiveCR"]  # 运行所有自适应参数的组合
```

### Q3: 如何只运行启用局部搜索的组合？

**A**: 使用关键词过滤：

```python
algorithms=["withLS"]  # 运行所有启用局部搜索的组合
```

### Q4: 如何组合多个过滤条件？

**A**: 使用多个关键词：

```python
algorithms=["rand/1", "adaptiveF", "withLS"]  # 运行同时包含这三个关键词的组合
```

### Q5: 如何知道某个组合的参数配置？

**A**: 查看算子组合名称，例如：

```
DE_rand/1_adaptiveF_bin_adaptiveCR_withLS
```

对应的参数：
- 变异策略：rand/1
- 自适应F：True
- 交叉策略：bin
- 自适应CR：True
- 局部搜索：True

### 示例3：自定义配置

```python
# custom_algorithm_configs.py
def generate_custom_algorithm_configs():
    configs = []
    
    # DE算法：只保留自适应F的组合
    for mut_strat in ["rand/1", "best/1", "adaptive"]:
        for cross_strat in ["bin", "exp"]:
            config_name = f"DE_{mut_strat}_adaptiveF_{cross_strat}_fixedCR_noLS"
            configs.append((config_name, "DE", {
                "mutation_strategy": mut_strat,
                "use_adaptive_F": True,
                "crossover_strategy": cross_strat,
                "use_adaptive_CR": False,
                "use_local_search": False,
            }))
    
    # GA算法：只保留锦标赛选择
    for cross in ["two_point", "hybrid"]:
        for mut in ["adaptive"]:
            config_name = f"GA_tournament_{cross}_{mut}_heuristic_noLS_noNeighborhood"
            configs.append((config_name, "GA", {
                "selection_strategy": "tournament",
                "crossover_strategy": cross,
                "mutation_strategy": mut,
                "initialization_strategy": "heuristic",
                "use_local_search": False,
                "neighborhood_size": 0,
            }))
    
    return configs

# main.py
exp_config = ExperimentConfig(
    instances=instance_files,
    algorithms=["custom"],  # 使用自定义配置
    seeds=[0, 1],
    deadlines=deadlines,
    max_evaluations=1000,
    output_dir="results/raw",
    time_limit=60.0,
    problem_type="rlp"
)
```

## 📌 注意事项

1. **修改main.py后需要重新运行程序**
2. **自定义配置需要修改runner.py导入自定义函数**
3. **关键词过滤是大小写敏感的**
4. **建议先测试少量组合，确认无误后再运行所有组合**
5. **保存好实验结果，避免重复运行**
